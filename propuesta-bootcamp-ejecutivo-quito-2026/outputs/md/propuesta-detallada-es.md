## 19. Glosario

| Termino | Definicion |
|---------|-----------|
| **Agente IA** | Software que ejecuta tareas autonomamente usando su contexto (correos, documentos, calendario) |
| **Pristino** | Agente Open Source de MetodologIA. Corre en local, auditable, suyo para siempre |
| **Asistente IA** | Herramienta especializada para una funcion especifica (ej: redactar correos, analizar datos) |
| **Segundo cerebro** | Sistema que organiza y conecta todo su conocimiento (basado en NotebookLM) |
| **Prompt** | Instruccion en lenguaje natural que le dice a la IA que hacer |
| **Bootcamp** | Programa intensivo de co-creacion (no curso teorico) |
| **Sistema agentico** | Conjunto coordinado de agente + asistentes + cerebro trabajando juntos |
| **CAO** | Chief Agentic Officer — directivo que opera con sistema de IA personal |
| **(R)Evolucion** | Filosofia MetodologIA: cambio con intencion, no con intensidad |
| **Embajador** | Graduado CAO certificado para facilitar programas MetodologIA |
