## 4. Alcance y exclusiones

### 4.1 Alcance incluido

1. Nivelar lenguaje comun y fundamentos de Microsoft Copilot en el entorno Microsoft 365.
2. Construir sistema de documentos profesionales con Word + Copilot.
3. Construir sistema de analisis de datos con Excel + Copilot (incluye dashboard interactivo).
4. Construir sistema de presentaciones profesionales con PowerPoint + Copilot.
5. Configurar sistema de gestion de email optimizado con Outlook + Copilot.
6. Configurar sistema de notas inteligentes con OneNote + Copilot.
7. Crear flujos de trabajo automatizados con Power Automate + Copilot.
8. Configurar ecosistema de colaboracion empresarial con Teams, OneDrive, SharePoint y Planner.
9. Ejecutar proyecto final integrador con presentacion y retroalimentacion.
10. Entregar materiales: 50 prompts ofimaticos, 1 playbook de implementacion, memorias, grabaciones y certificados.

### 4.2 Exclusiones explicitas

1. Creacion de asistentes personalizados, GPTs o configuracion de Copilot Studio.
2. Consultoria de negocio, estrategia organizacional o transformacion digital.
3. Implementacion tecnica: APIs, integraciones, desarrollo de software.
4. Administracion de tenant Microsoft 365 o configuracion de licencias Copilot.
5. Soporte tecnico de Microsoft 365 o Copilot (responsabilidad de Microsoft o partner del cliente).
6. Ejecucion de tareas del participante.
7. Decisiones legales o contractuales.

## 7. Obligaciones del cliente

1. El cliente debera proveer cuentas Microsoft 365 (Business o Enterprise) activas para cada participante con al menos 5 dias habiles de anticipacion.
2. El cliente debera asegurar licencias de Microsoft Copilot asignadas y activas en cada cuenta con al menos 5 dias habiles de anticipacion.
3. El cliente debera garantizar que las politicas de TI no bloquean Microsoft Copilot con al menos 3 dias habiles de anticipacion.
4. El cliente debera confirmar el listado de participantes con al menos 48 horas antes de la primera sesion.
5. El cliente debera asegurar la disponibilidad de los participantes durante las 18 horas programadas (9 sesiones de ~2h).
6. El cliente debera completar el brief de contextualizacion con al menos 5 dias habiles de anticipacion (si desea ejercicios personalizados).
